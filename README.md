## Overview
TechTree is a skill visualization app using graph structure.

## Tech Stack
- Frontend: Angular, Cytoscape.js
- Backend: Django REST Framework
- DB: PostgreSQL

## How to Run
(frontend)
npm install
ng serve

(backend)
pip install -r requirements.txt
python manage.py runserver
